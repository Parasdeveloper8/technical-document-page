It is maintained by Meta and a community of individual developers and companies. React can be used to develop single-page, mobile, or server-rendered applications with frameworks like Next.js.

Angular

Angular is a TypeScript-based free and open-source single-page web application framework run on Node.js.

It is developed by Google and by a community of individuals and corporations. Angular is a complete rewrite from the same team that built AngularJS.

  1. current version 18.0.0

Vue

Vue.js is an open-source model–view–viewmodel front end JavaScript framework for building user interfaces and single-page applications.

It was created by Evan You, and is maintained by him and the rest of the active core team members.

  1. current version 3.4.27
import { createApp } from 'vue' import App from './App.vue' const app = createApp(App)

JQuery

jQuery is a JavaScript library designed to simplify HTML DOM tree traversal and manipulation, as well as event handling, CSS animations, and Ajax.

It is free, open-source software using the permissive MIT License. As of August 2022, jQuery is used by 77% of the 10 million most popular websites.

  1. current version 3.7.1
var hiddenBox = $( "#banner-message" ); $( "#button-container button" ).on( "click", function( event ) { hiddenBox.show(); });

Express

Express.js, or simply Express, is a back end web application framework for building RESTful APIs with Node.js, released as free and open-source software under the MIT License.

It is designed for building web applications and APIs. It has been called the de facto standard server framework for Node.js.

  1. current version 4.19.2
